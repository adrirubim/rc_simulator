## Icon assets

### Source of truth

- `rc-simulator.svg`: canonical vector icon (used on Linux and as the source for raster exports).

### Windows

- `rc-simulator.ico`: icon for Windows shortcuts (`.lnk`). Windows Explorer does not support SVG icons for shortcuts.
